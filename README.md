# Airbnb Clone Project Documentation

This repository contains documentation for the Airbnb Clone backend project.  
It includes diagrams, user stories, and requirements that describe the system design and backend structure.

## Overview
The Airbnb Clone backend is built to manage users, properties, bookings, payments, and reviews.  
The goal is to create a secure, scalable, and efficient system similar to Airbnb.

## Contents
- **features-and-functionalities/** – Project features and requirements  
- **use-case-diagram/** – Use case diagram showing system interactions  
- **user-stories/** – Core user stories from the use case diagram  
- **ERD/** – Entity Relationship Diagram  
- **normalization/** – Database normalization explanation
